<p>Ubuntu 16.04 LTS の box を使用する。</p>
<p>Vagrantfile の作成。</p>
<pre><code class="hljs language-bash">$ vagrant init ubuntu/xenial64
</code></pre>
<p>作成した Vagrantfile のポートを設定。</p>
<pre><code class="hljs language-bash">$ vim Vagrantfile
</code></pre>
<p>下記の部分を変更。</p>
<pre><code class="hljs language-vagrantfile">config.vm.network "forwarded_port", guest: 8000, host: 8000
</code></pre>
<p>起動。</p>
<pre><code class="hljs language-bash">$ vagrant init ubuntu/xenial64
</code></pre>
<p>接続。</p>
<pre><code class="hljs language-bash">$ vagrant ssh
</code></pre>
<h2>Django のインストール</h2>
<p>pip のインストール。</p>
<pre><code class="hljs language-bash">$ wget https://bootstrap.pypa.io/get-pip.py
$ <span class="hljs-built_in">sudo</span> python3 get-pip.py
$ <span class="hljs-built_in">sudo</span> pip install Django
</code></pre>
<p>Ubuntu 16.04 LTS にはデフォルトで Python3.5.2 が入っている。</p>
<p>バージョンの確認。</p>
<pre><code class="hljs language-bash"><span class="hljs-variable">$python3</span> -m django --version
1.11.5
</code></pre>
